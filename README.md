# HW1 – Coding Assistant Onboarding

This repository contains my submission for Homework 1 in Generative AI, where I set up a coding assistant, created draft content, edited it manually, and tracked the whole workflow with Git and GitHub.

## Git workflow used

- Created the repository and added the initial files (`README.md`, `assistant_draft.md`, `final_version.md`).
- Used OpenCode in the project folder to generate the first version of `assistant_draft.md`.
- Copied the assistant draft to `final_version.md` and made my own manual edits without using the assistant.
- Committed each stage separately and pushed the repository to GitHub after updating the README with the video link.
